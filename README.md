Social Media App - CSS Frameworks Branch
Overview

This project is a social media app prototype built with a CSS framework called Tailwind on a dedicated branch named css-frameworks. It features responsive UI design for the following pages:

Authentication page (/index.html): Login and registration form with validation.

Feed page (/feed/index.html): List of posts with thumbnails, search bar, sorting options, and new post creation form.

Profile page (/profile/index.html): User profile with profile image, username, posts, follow button, and follower/following info.


The project aims to deliver a fully responsive UI meeting the specified criteria without advanced JavaScript
