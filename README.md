# Social Media App – CSS Frameworks Branch

## Overview

This project is a social media app prototype built using Tailwind CSS on a dedicated branch called `css-frameworks`.

The goal of the assignment was to create a responsive user interface using a CSS framework, without focusing on advanced JavaScript functionality.

The project includes three main pages and demonstrates layout structure, responsive design, and consistent styling.

---

## Pages

### Authentication Page (/index.html)
- Login and registration form
- Email validation pattern
- Responsive centered layout
- Styled using Tailwind utility classes

### Feed Page (/feed/index.html)
- List of posts with images
- Search input and sorting dropdown (UI only)
- Create post form
- Responsive layout using Flexbox
- Active navigation link styling

### Profile Page (/profile/index.html)
- Profile image and user information
- Follow button
- Follower and following statistics
- List of user posts
- Responsive layout for mobile and desktop
